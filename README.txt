# Accessible SFC

### 視覚障害者もバス時刻割が分かるように、次のバスの時刻を読み上げるアプリ。
このプロジェクトは、目が見えないグループメンバーのため始まったもので、個人利用目的で作られましたが、現在は[SFC Bustimer](https://github.com/leochoo/SFC-Bustimer)アップグレードされました。

### A text-to-speech app that reads bus information for visually impaired people.
This project was only targetted at my friend who is visually impaired but the next, upgraded version is available as an iOS app [SFC Bustimer](https://github.com/leochoo/SFC-Bustimer)


### 機能
* ボタンを押すと、現時刻と次のバスまでの残り時間を読み上げる
* SFC <-> 湘南台駅のバスの時刻情報

### Features
* Reads the current time and the time left until the next bus
* Bus schedules of SFC <-> Shonandai
